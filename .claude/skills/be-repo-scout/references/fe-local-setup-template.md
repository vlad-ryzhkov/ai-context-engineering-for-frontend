## 2. Quick Start (5 Steps)

```bash
# 1. Clone and configure env
cp .env.example .env
# Edit .env — set required vars (see §3 below)

# 2. Start backend services
docker-compose up -d

# 3. Verify service is running
curl http://localhost:{PORT}/health

# 4. Obtain auth token (if auth required)
curl -X POST http://localhost:{PORT}/auth/login \
  -H "Content-Type: application/json" \
  -d '{"email": "{test_email}", "password": "{test_password}"}'

# 5. Make a test request
curl http://localhost:{PORT}/{first_endpoint} \
  -H "Authorization: Bearer {token_from_step_4}"
```
